Homelab

A tidy, segmented network with observability and guardrails.

Topology

Core: Netgear GS305E, VLAN1‑3; WAN → router → switch → trunk to servers.

Servers: PC1 & PC2 (Ubuntu), static IPs 192.168.1.x, Docker services.

Security: Suricata (IDS/IPS), EveBox, Wazuh, AdGuard Home.

Highlights

  • DNS‑level ad‑blocking via AdGuard Home
  • VLAN segmentation for lab vs LAN
  • Egress controls, alerts to dashboards
  • Containerized services managed via Compose

Key Services

AdGuard Home

DNS sinkhole, rewrites, client policies.

docker compose -f adguard.compose.yml up -d

Suricata + EveBox

Custom rules, PCRE, noisy rule suppression.

suricata -c /etc/suricata/suricata.yaml -i eth0

Wazuh

Host IDS + SIEM, file integrity, auth events.

systemctl status wazuh-manager
